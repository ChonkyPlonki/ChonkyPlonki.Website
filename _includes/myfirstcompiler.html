How the Compiler Works:

  1. Parsing: Parses code words and numbers into calculations.
  2. Shunting Yard Algorithm: Translates these into an ordered instruction list via the Shunting Yard Algorithm.
  3. Assembly Code Translation: Translates the ordered instructions into assembly code that can then be run on a Windows x64 machine.
The code quality I created is questionable (and so is the lack of tests), but the journey, the learning, and the fun were the main focus.
