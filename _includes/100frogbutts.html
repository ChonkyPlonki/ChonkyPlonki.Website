Quick Info

  • Project Type: Hidden Object Game
  • Role: Solo Project
  • Game Engine: Unity
  • Status: Not Completed
A Hundred Frog Butts is a silly hidden object game where you have to find hidden frogs in a scene by searching for their butts.
I was inspired by humorous Instagram and TikTok videos where artists would paint or sculpt cute fruits or animals (like this one for example of a beet), then adjust the angle to show they had added a little sculpted butt to them too.
So this game was aimed to be a game-version of those silly videos!
