THE DUCK POND is an idle clicker game I made together with the awesome Jonathan Hertz. We released it on Steam 26 July, 2024.

We set out to make a minimal game and complete it from start to finish within 1-2 months. We were both working full-time at the time so it was very challenging, but we did it and it was worth it!
We ended up having the game as "To be released" on Steam for only 2-3 weeks before publishing, and only marketed it for that short timem so we were very happy by how positive the reception was.
